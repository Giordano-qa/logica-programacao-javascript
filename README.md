# 🚀 Desafios de Lógica de Programação com JavaScript

Este repositório contém uma série de 15 exercícios práticos de lógica de programação desenvolvidos em **JavaScript (Node.js)** durante a minha formação em QA na **Lumitek**. 

O objetivo deste projeto foi consolidar os fundamentos da programação, essenciais para a construção de automações de testes robustas e eficientes.

## 🛠️ Tecnologias Utilizadas
* **JavaScript (ES6+)**
* **Node.js** (ambiente de execução)
* **VS Code** (IDE)

## 🧠 Conceitos Praticados
* Manipulação de variáveis e tipos de dados (Strings, Numbers, Arrays).
* Estruturas condicionais avançadas (`if`, `else if`, `else`).
* Laços de repetição e iterações (`for`, loops de varredura de strings).
* Criação de funções modulares e reutilizáveis com `return`.
* Operadores lógicos e matemáticos (como o operador de resto da divisão `%`).

## 📋 Principais Desafios Resolvidos
* **Verificar se o Número é Primo:** Uso de laços estruturados e controle de fluxo com múltiplos retornos.
* **Contar Vogais em uma Frase:** Manipulação de strings, métodos nativos (`.toLowerCase()`, `.includes()`) e contagem acumulada.
* **Verificar se número é Par ou Ímpar:** Aplicação de condicionais com o operador módulo (`%`).
